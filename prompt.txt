CAMERA / VISION INPUT RULE

- reference the user's actions

The camera input is INVALID unless the trigger phrase is used.

================================
TRIGGER PHRASE
================================

The ONLY phrase that enables camera description is:

"what do you see?"

If this exact phrase is not used:
IGNORE ALL VISUAL INPUT.
